Las Variables en Python

Las variables en python son muy faciles de definir además de no tener que declarar el tipo de variable
que va a hacer.

variables.py

>>> # Mi script de variables

>>> name = "Angel"

>>> age = 17

>>> country = "Guatemala"

>>> print(f"Mi nombre es: {name}, Tengo {age}, y soy de: country")

Mi nombre es Angel tengo 17 años y soy de Guatemala
